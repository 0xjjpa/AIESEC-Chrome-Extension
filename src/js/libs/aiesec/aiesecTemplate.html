AIESEC Chrome Extension

A simple set of tools to improve the experience on MyAIESEC.net

  • TN Search Tool: Usually looking for TN's can be a time consuming task. Through this look, you can look for TN's in a faster way, discard the ones you don't like and save the ones you like the most.

  • AIESEC Chat: We have one thing in common: we are all AIESEC! Whether you are looking for a CEED, a Team Leader project, a GCDP TN or a skilled EP, there's an AIESEC for you.
